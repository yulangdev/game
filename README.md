# 게임 맵 에디터 (2010년 대학교 졸업작품)

높낮이가 있는 마름모꼴 타일 맵 에디터.

주변 지형 타일의 종류(잔디/흙)에 따라 자동 매핑. (스타크래프트 맵 에디터 방식)

* 이미지 : 파랜드 택틱스 2
